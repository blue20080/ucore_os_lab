# 背景

本人为一名已经参加工作5+年的IT从业者，从事大数据相关的开发工作，并没有系统的学习过Computer Sicence知识。在工作中涉及一些代码编码、系统调优从网上相关的资料，只能知道How（怎样做），但不知道Why（为何这样做）。故系统的学习操作系统知识。



# 学习目标

掌握OS设计实现、理解操作系统工作原理

掌握内存管理、进程\线程管理、文件系统设计和实现
